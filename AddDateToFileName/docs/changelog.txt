# Changelog for Add Date To Filename  #
####Adds the date the file was created to the filename before the suffix.
## v1.1 - 27 Mar 2013 ##
* Default date format changed to YYYYMMDD per advice from [atadams](http://cloud.zoooot.com/NrE6).
Item Name.xxx -> Item Name-20130523.xxx

##v1.0 - 25 Mar 2013
* Initial release
Select a file in Finder and hit your hotkey.
E.g. Item Name.xxx -> Item Name-03-25-2013.xxx
